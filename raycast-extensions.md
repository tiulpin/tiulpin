## Notes

Raycast settings including hotkeys, API keys, and preferences are stored in:
- `~/Library/Preferences/com.raycast.macos.plist`
- `~/.config/raycast/config.json` (contains access tokens - DO NOT commit)

To restore extensions, install them manually from Raycast Store or via the web interface.
