> [!NOTE]
> 来自 [InsideTheRusticKitchen](https://www.insidetherustickitchen.com/parmigiana-di-melanzane-aubergine-parmigana/)。Parmigiana di Melanzane 是一道终极舒适食谱。用番茄酱、罗勒、帕尔马干酪和马苏里拉干酪烘烤油炸茄子/鸡蛋。

| **产品** | **数量** |
|---|---|
| 茄子/茄子泥 | 2 个大的（约 700 克/1.5 磅） |
| 番茄泥（passata），块状或滑溜状 | 1.5 磅（700 克） |
| 白洋葱 | 1 颗 |
| 罗勒 | 一小束 |
| 马苏里拉奶酪，切块 | 8.8 盎司（250 克） |
| Parmigiano Reggiano 芝士，新鲜磨碎 | ¾ 杯（70 克） |
| 橄榄油 | ½ 汤匙 |
| 面粉，用于撒粉 | 视需要而定 |
| 葵花籽油，煎炸用 | 约 ½ 杯（125 毫升） |
| 盐和胡椒调味 | 适量 |

### 准备工作

1. **准备茄子：**
    - 将茄子切成薄片，放入砂锅中。撒上少许盐，搁置 1 小时，以去除多余水分，帮助油炸。

2. **番茄酱：**
    - 切碎洋葱，在平底锅中加入橄榄油炒至半透明，变软，但不变色。
    - 加入番茄酱和撕碎的罗勒叶。加入盐和胡椒调味。搅拌后小火慢炖 10-15 分钟，然后搁置一旁。

3. **煎茄子：**
    - 将茄子片在冷水中冲洗干净，用毛巾拍干。轻轻撒上面粉，抖掉多余的面粉，然后在葵花籽油中每面煎几秒钟。在厨房纸上沥干多余的油。

4. **组装帕尔马干酪：**
    - 烤箱预热至 180°C（350°F，煤气标度 4）。使用 11×8 英寸的椭圆形盘子或类似大小的盘子。
    - 在盘子底部涂抹少量番茄酱，以防粘底。
    - 铺上茄子片，撒上帕尔马干酪，加入马苏里拉奶酪块，用胡椒调味，再加入几勺番茄酱。保留 ⅓ 的酱汁用于最上层。
    - 重复分层，直到剩下一层。

5. **最后一层和烘烤：**
    - 用勺子将剩余的番茄酱淋在最上层，最后淋上帕马森芝士和马苏里拉芝士。
    - 盖上锡纸，烘烤 20 分钟。揭开锡箔纸，再烤 20 分钟，直至表面金黄起泡。
    - 上桌前静置 5 分钟。
