> [!NOTE]
> 土耳其鸡蛋在制作完成后最好立即品尝，此时水煮蛋温热，香料黄油酱芳香四溢。水煮时要确保水温在小火慢炖的状态，这样鸡蛋才能完美凝固。

| **产品** | **数量** |
|---|---|
| 希腊酸奶（原味，全脂） | 250-300 克 |
| 鸡蛋 | 4 只大鸡蛋 |
| 新鲜大蒜 | 1-2 瓣（每瓣≈ 3-4 克） |
| 白醋 | 5-10 毫升（1-2 茶匙） |
| 无盐黄油 | 45-60 克（≈ 3-4 汤匙） |
| 辣椒粉 | 2-3 克（≈ ½ 茶匙） |
| 红辣椒片 | 一小撮（≈ 0.25-0.5 克） |
| 新鲜莳萝或欧芹 | 10-15 克（切碎，用于点缀） |
| 特级初榨橄榄油 | 5 mL（≈ 1 茶匙） |

### 制作方法

1. **准备酸奶：**
   在碗中加入希腊酸奶、切碎的新鲜大蒜和少许盐。静置几分钟，让大蒜的香味渗入其中。

2. **水煮鸡蛋：**
    - 将一锅水煮沸（约 82°C-88°C）。
    - 在水中加入白醋，帮助蛋白凝固。
    - 将每个鸡蛋打入一个小杯中，然后一个一个轻轻放入沸水中。
    - 水煮鸡蛋 3-4 分钟，直到蛋清凝固，但蛋黄仍然是流动的。用漏勺捞出，放在纸巾上沥干。

3. **制作香料黄油酱：**
    - 在一个小平底锅中，用中火融化无盐黄油。
    - 加入辣椒粉和红辣椒片搅拌。煮约 1 分钟，直至酱汁变香。离火。

4. **装盘：**
    - 将蒜味酸奶铺在盘子上。
    - 将水煮蛋放在酸奶上。
    - 在鸡蛋上淋上温热的香料黄油酱。
    - 用切碎的新鲜莳萝或欧芹和少许特级初榨橄榄油点缀。

5. **上桌：**
   立即享用土耳其鸡蛋，以获得最佳的口味和口感！
