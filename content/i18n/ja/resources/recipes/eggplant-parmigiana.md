> [!NOTE]
> [InsideTheRusticKitchen](https://www.insidetherustickitchen.com/parmigiana-di-melanzane-aubergine-parmigana/)より。パルミジャーナ・ディ・メランツァーネは究極のコンフォートフード・レシピ。揚げ茄子をトマトソース、バジル、パルメザンチーズ、モッツァレラチーズと一緒に焼いたもの。

| **材料** | **分量** |
|---|---|
| ナス | 大2本（700g前後） |
| トマトのピューレ（パッサータ）（かため、またはなめらかに） | 1.5ポンド（700g） |
| 白たまねぎ | 1個 |
| バジル | 小房 |
| モッツァレラチーズ（角切り） | 250g |
| パルミジャーノ・レッジャーノ・チーズ（すりおろし） | 3/4カップ（70g） |
| オリーブオイル | 大さじ1/2 |
| 小麦粉 | 適量 |
| ひまわり油（揚げ物用） | ½カップ（125ml） |
| 塩・こしょう | お好みで |

### 作り方

1. **ナスを準備する:**
   - ナスを薄くスライスし、ザルに上げる。軽く塩をふり、1時間ほど置いておく。

2. **トマトソース:**
   - 玉ねぎをみじん切りにし、オリーブオイルをひいたフライパンで半透明になるまで炒める。
   - トマトパッサータとちぎったバジルの葉を加える。塩と胡椒で味を調える。かき混ぜながら10～15分軽く煮込んだら、脇に置いておく。

3. **ナスを炒める:**
   - 薄切りにしたナスを水洗いし、タオルで水気を拭き取る。小麦粉を薄くまぶして余分な粉を振り落とし、ひまわり油で両面を数秒ずつ揚げる。キッチンペーパーの上で余分な油を落とす。

4. **パルミジャーナを作る:**
   - オーブンを180℃に予熱する。11×8インチの楕円形の皿か、同じような大きさの皿を使う。
   - 皿の底にトマトソースを少量塗り、くっつかないようにする。
   - ナスの薄切りを重ね、パルメザンチーズをふりかけ、角切りにしたモッツァレラチーズを加え、胡椒で味を調え、トマトソースをスプーン2～3杯加える。ソースの⅓は上の層用に取っておく。
   - 層が1つ残るまで繰り返す。

5. **最後の層と焼く:**
   - パルメザンチーズとモッツァレラチーズで仕上げる。
   - ホイルでゆるく覆い、20分焼く。ホイルを外し、表面がこんがりとするまでさらに20分焼く。
   - 食べる前に5分休ませる。
