> [!NOTE]
> [キヌア入りタブーリ](https://en.wikipedia.org/wiki/Tabbouleh) - 爽やかな中東料理。

| **材料** | **分量** |
|---|---|
| キヌア | 1カップ |
| 水 | 2カップ |
| トマト | 10個以上（みじん切り） |
| パセリ | 3/4カップ（みじん切り） |
| ミント | 3/4カップ（みじん切り） |
| ネギ | 4本（みじん切り） |
| レモン汁 | 大さじ2杯 |
| オリーブオイル | 大さじ1 |
| 胡椒 | 小さじ1 |

### 作り方

1. キヌアを水洗いする。カップ2杯の水を沸騰させ、キヌアを入れ、柔らかくなるまで15分煮る。冷ます。

2. 大きめのボウルに、冷ましたキヌア、細かく刻んだローマトマト、パセリ、ミント、ねぎ、レモン汁、オリーブオイル、こしょうを入れる。よく混ぜる。

3. 冷やして、または室温でいただく。
