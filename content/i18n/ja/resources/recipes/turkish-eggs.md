> [!NOTE]
> トルコ風エッグは、ポーチドエッグが温かく、スパイスの効いたバターソースが香ばしいうちにお召し上がりください。ポーチドエッグを作るときは、お湯を沸騰させないように注意しましょう。

| **材料** | **分量** |
|---|---|
| ギリシャヨーグルト（プレーン、全脂肪） | 250-300g |
| 卵 | 大4個 |
| ニンニク | 1～2片（1片あたり3～4g） |
| 白酢 | 5～10mL（ティースプーン1～2杯） |
| 無塩バター | 45～60g（≒大さじ3～4） |
| パプリカパウダー | 2～3g（≒小さじ1/2） |
| レッドペッパーフレーク | ひとつまみ（≒0.25～0.5g） |
| ディルまたはパセリ | 10～15g（みじん切り、飾り用） |
| エクストラバージンオリーブオイル | 5 mL（≒小さじ1） |

### 作り方

1. **ヨーグルトを準備する:**
   ボウルにギリシャヨーグルトとみじん切りにしたにんにく、塩をひとつまみ入れる。ニンニクの風味を染み込ませるため、数分間置く。

2. **卵を茹でる:**
   - 鍋に湯を沸かし、弱火にかける（約82℃～88℃）。
   - 卵白を凝固させるために、ホワイトビネガーを水に加える。
   - 小さなカップに卵を割り入れ、沸騰したお湯の中に1個ずつ静かに入れる。
   - 白身は固まるが黄身はふやけるまで、3～4分間ゆでる。スプーンで取り出し、ペーパータオルの上で水気を切る。

3. **スパイスバターソースを作る:**
   - 小さな鍋に無塩バターを入れ、中火で溶かす。
   - パプリカパウダーとレッドペッパーフレークを混ぜる。ソースに香りが出るまで約1分加熱する。火からおろす。

4. **皿に盛り付ける:**
   - にんにく風味のヨーグルトを皿に広げる。
   - ヨーグルトの上にポーチドエッグをのせる。
   - 温かいスパイスバターソースをかける。
   - 刻んだフレッシュ・ディルまたはパセリを飾り、エキストラ・バージン・オリーブオイルを軽くかける。

5. **お召し上がり方:**
   トルコ風エッグはすぐにお召し上がりください！
